Sobre la materia

DESARROLLO WEB I

Descripción

La materia abarca los conceptos básicos de HTML, CSS y JavaScript. Mediante los cuales los estudiantes serán capaces de crear páginas web atractivas y prácticas, aplicaciones interactivas de front-end y comprender la arquitectura de las mismas

Función clave

Construir soluciones a través de algoritmos y procedimientos informáticos, utilizando las herramientas de programación de software para resolver problemas en diferentes ámbitos

Contenidos

  • Unidad I: Introducción al HTML
  • Unidad II: Introducción a las hojas de estilo en cascada
  • Unidad III: Maquetación de sitios web
  • Unidad IV: Frameworks de diseño web responsivo
  • Unidad V: Introducción a JavaScript
  • Unidad VI: El modelo de objetos del documento (DOM) y JavaScript asíncrono

Metodología

  • Lectura previa y visualización de materiales audiovisuales
  • Sesiones de trabajo presencial
  • Actividades no evaluadas de aprendizaje
  • Sesiones en línea consistentes en clases y ejercicios autodirigidos
  • Proyecto grupal

Mis trabajos

GUÍAS

Guía 1: Introducción a HTML y CSS

Se identificó la estructura básica de un documento web, se aplicaron las reglas de estilo CSS a los elementos HTML usando estilos embedidos y hojas de estilo externas, y se utilizaron los selectores básicos y propiedades CSS para fuentes y textos

Estilos Embedidos Julio Cortazar Países Ejercicio Serie TV

Guía 2: Selectores CSS avanzados y Modelo de caja

Se utilizaron y combinaron selectores CSS avanzados, se aprendió sobre la utilidad de las pseudo-clases y pseudo-elementos CSS, y se creó una página web utilizando el Modelo de caja e identificando sus respectivas partes

Película Tableto Ejercicio Mi Biografía

Guía 3: Posicionamiento, visualización y maquetación de sitios web

Se utilizaron las propiedades CSS relacionadas con la visualización de elementos y los diferentes tipos de posicionamiento CSS, y se maquetaron sitios web con diseños fijos y fluidos

Óscares Menú Horizontal Viajeros Ejercicio Cartas

Guía 4: Introducción al Responsive Design

Se crearon y estilizaron formularios utilizando los controles de HTML5, se utilizó la meta-etiqueta viewport, se definieron las medias queries para aplicar reglas de estilo en función del ancho de viewport de navegador y se utilizaron los conocimientos adquiridos en la asignatura para la construcción de páginas web responsivas

Media Queries Ejercicio Portafolio Estudiantil

Guía 5: Introducción a Javascript

Se adquirió dominio de los elementos sintácticos básicos del lenguaje Javascript, se generó codigo HTML desde secuencias de comando de JS, se crearon páginas web utilizando métodos básicos de interacción con el usuario y se utilizaron correctamente las estructuras selectivas y repetitivas en JS

Interactuando con JavaScript

Guía 6: Funciones y eventos en JS

Se adquirió dominio en la construcción de funciones con JavaScript, se utilizaron parámetros o argumentos en las funciones y se aprendió sobre el manejo de eventos mediante funciones como controladores de eventos asociados a elementos del documento web

Estructuras de Control y Arreglos Funciones

Guía 7: Document Object Model (DOM)

Se adquirió un dominio amplio del DOM; la capacidad de acceder, eliminar y modificar el contenido de una página web a través del acceso a los nodos que forman el documento; y, se emplearon los métodos y propiedades del estándar DOM para tener un dominio completo sobre el documento HTML

ANÁLISIS FODA

Fortalezas

  • Dominio sólido de los elementos, semántica y atributos HTML
  • Interés en el diseño visual y la estética, lo que favorece mi desempeño en las unidades de CSS, layout y diseño responsivo
  • Comprensión profunda del Modelo de caja, posicionamiento y maquetación
  • Capacidad para crear diseños responsivos
  • Habilidad para comprender conceptos lógicos de JavaScript

Oportunidades

  • Aprender HTML, CSS y JavaScript me abre la posibilidad de desarrollar sitios web complejos y mejorar mi empleabilidad
  • Tener conocimientos en Bootstap, Flexbox y CSS Grid me permite adaptarme a los estándares modernos del diseño web responsivo
  • Dominio del DOM y JSON que me permiten desarrollar aplicaciones web interactivas
  • Habilidades transferibles a diseño UX/UI
  • Con lo aprendido, a lo largo de la materia, tengo la capacidad para trabajar en proyectos relacionados con el desarrollo web

Debilidades

  • Una mayor práctica en la integración completa de HTML, CSS y JS dentro de un mismo proyecto
  • Profundizar en la lógica avanzada de JavaScript (asincronía, manejo de APIs y almacenamiento local)
  • Conocimiento limitado de patrones de diseño avanzados
  • Mejorar la organización del código y estructura de archivos

Amenazas

  • Limitaciones de tiempo para practicar fuera de la materia
  • Falta de experiencia en proyectos reales con clientes
  • Dificultad para mantenerme actualizada con nuevas herramientas
